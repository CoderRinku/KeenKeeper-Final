# 👥 KeenKeeper

## 📖 What is this project?

KeenKeeper is a simple web app built to help us keep our friendships alive. In our busy lives, we often forget to check in on our close ones. This dashboard works as a personal reminder to nurture those meaningful connections, tracking when we last talked and setting goals to stay in touch.

## 🛠️ Tools & Technologies I Used

- **Frontend:** Next.js and React.js
- **Styling:** Tailwind CSS (made the responsive design super easy)
- **Charts:** Recharts (used for the analytics donut chart)
- **Icons & Alerts:** React Icons and React Hot Toast
- **Database/Storage:** Used LocalStorage to make the timeline interactions work dynamically

## ✨ 3 Cool Features

1. **Friend Dashboard:** 
A clean grid layout showing all my friends, with color-coded tags indicating if a catch-up is "Overdue" or "On-track".

2. **Working Timeline:**
 When I click "Call", "Text", or "Video" on a friend's details page, it actually saves the interaction. I can then see and filter all these logs on the Timeline page!

3. **Analytics Chart:** 
A beautiful donut chart on the Stats page that visually breaks down my interaction habits (how many times I texted vs called).
